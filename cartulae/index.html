Keyboard Layouts

These keyboard layouts for Mac OS X are free to use and distribute.
To install, unzip the file in ~/Library/Keyboard Layouts then add them via System Preferences > Keyboard > Input Sources.
To see how the characters are mapped, go to System Preferences > Keyboard > Input Sources and check "Show input menu in menu bar". From the input menu, click on "Show Keyboard Viewer".

  • Ethiopic [Download]
    • Character forms change as you type. Pressing [q] produces ቅ, which changes to ቃ if you press [a] immediately afterwards.
    • Labialized characters can be accessed using the [option] key.
    • Transliterated characters (upper- and lowercase) are available using [caps lock].
    • Some Amharic and Tigrinya characters are also available using the [option] key.
  • Coptic [Download]
    • Largely based on Mac OS' included Greek (Polytonic) keyboard. Some characters and diacritics have been changed accordingly. The full character set can be accessed using the [option] and [shift] keys.
    • Characters unique to particular dialects (such as Akhmimic) are not available.
