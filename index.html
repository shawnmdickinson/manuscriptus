About

Ms: Codex is a free, elegant application for organizing and analyzing manuscript corpuses. It is open-source (MIT License), which means that it is free to use, to modify, and to distribute. It uses powerful, cross-platform technologies and features a simple interface. It also has a plugin API for adding more features.

FAQ

If Ms: Codex is web-based, does that mean I need a website? What if I want to use it offline?
Ms:Codex can be run locally so that only you can use it and no internet connection is required. The interactive getting started guide has instructions for setting up Ms: Codex offline.
Does Ms: Codex support non-Latin characters?
Yes! It uses UTF-8 (Unicode) encoding, meaning that any characters that are supported by UTF-8 (Latin, Greek, Cyrillic, Arabic, Ethiopic, etc.) are supported. Note that you will need to have appropriate fonts on your computer for these characters.
Can Ms: Codex be used collaboratively?
If Ms: Codex is set up on a server (rather than locally), then anyone with access to the server can work on it. Currently, a plugin is in the works for adding user accounts to control access.
