Manuscriptus Pro

Manuscriptus Pro is an open-source application for organizing and analyzing manuscript corpuses.

An open-source application built for the web.

Manuscriptus Pro is free, modifiable, modern, and cross-platform.

Open Source

Manuscriptus Pro is released under the MIT license, which means that it's free to use, free to modify, and free to distribute. It also comes with a plugin API for adding features and customizations.

Cutting-Edge

Manuscriptus Pro is built using powerful, modern, web-based technologies. Written in PHP, it uses a MySQL database with a Bootstrap front-end. It features an easy-to-use, intuitive interface optimized for quick workflow.

Cross-Platform

Because it's built for the web, Manuscriptus Pro can run on any computer with a browser and can be used online on a server or offline in a local environment. The setup is painless and the requirements are minimal.

FAQ

What are the minimum requirements?
PHP 5.0 (recommended 5.4); MySQL 5.0 (recommended 5.5); Safari, Chrome, Opera, or Internet Explorer; JavaScript enabled.
What languages are supported?
Manuscriptus Pro uses UTF-8 (Unicode) encoding, meaning that any characters that are supported by UTF-8 (Latin, Greek, Cyrillic, Arabic, Ethiopic, etc.) are supported. However, no font support is provided (see this page for more info).
What localizations are available?
Currently, the only language available for the interface is English. There are plans to offer official localizations in French and in German. If you're willing to work on a new localization (thank you!), please submit a feature request using GitHub's issue tracker.
